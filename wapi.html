WAPI

This project is to observe the air pollutant index in Westports working area.

The data is retrieved from API that were converted from IoT device that were integrated and handled by other web developers

The converted data was then integrated to mobile apps and displayed through mobile application

The user can observe the air pollutant index in designated area

I worked on this system for 3 months before continuing with other projects

Technologies:
  • - Flutter
  • - Dart
  • - API Integration
